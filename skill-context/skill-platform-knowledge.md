## X / Twitter

### Hard Limits

| Element                  | Limit                   |
| ------------------------ | ----------------------- |
| Tweet (standard account) | 280 characters          |
| Tweet (Premium/X Blue)   | 25,000 characters       |
| URL (always counts as)   | 23 characters           |
| Hashtags recommended     | 1–2 max                 |
| Thread sweet spot        | 5–10 tweets (7 optimal) |

### Optimal lengths (engagement data)

- Single tweet: **71–100 characters** (peak retweets); under 100 gets 17% more engagement
- Hard max for performance: stay under 200 characters in most cases
- Threads with images: 150% more retweets than text-only

### Algorithm signals (ranked by weight)

1. **Replies** — highest weight
2. **Quote tweets** — high weight
3. **Bookmarks / saves** — high weight
4. **Retweets** — medium weight
5. **Likes** — lowest weight

### Content rules

- Links in tweet body suppress reach → put links in **first reply**
- Threads: each tweet must work as a standalone unit
- Cliffhangers every 1–2 tweets maintain scroll momentum
- Hashtags go in the **last tweet** of a thread
- Real-time/trending content gets algorithmic boost

### Post formats

- Single tweet
- Thread (connected series)
- Quote tweet
- Poll (max 4 options, runs up to 7 days)
- Long-form post (Premium only, shown truncated in feed)

### Audience & tone

- Real-time, opinionated, direct
- Contrarian and counter-intuitive content performs well
- Avoid corporate language — personal voice wins
- Best for: thought leadership, breaking news, conversation

## Threads (Meta)

### Hard Limits

| Element              | Limit                          |
| -------------------- | ------------------------------ |
| Post character limit | 500 characters                 |
| Links                | Supported                      |
| Video length         | Up to 5 minutes                |
| API publishing limit | ~250 posts per 24h per profile |

### Optimal lengths

- Keep posts conversational: **150–300 characters** ideal
- Series of connected posts work like Twitter threads

### Algorithm signals

- **Replies** → +42% engagement boost (strongest signal)
- **Reposts + Quotes** → secondary signal
- Consistent posting rewards reach over time
- Mid-week mornings (Wednesday 7 AM) peak engagement window

### Content rules

- Tone: casual, conversational, less polished than Instagram
- Avoid corporate-sounding language
- Behind-the-scenes and authentic content performs best
- Don't just cross-post from X — rewrite natively
- Series and recurring content build audience anticipation
- External links: allowed but reduce organic reach; put in replies when possible

### Post formats

- Text post
- Image (single or carousel)
- Short video
- Poll
- Linked thread (series of connected posts)

### Audience & tone

- Skews male (57%), primarily 25–34
- Alternative to X: less toxic, more conversational
- Best for: community building, brand personality, dialogue

## Instagram

### Hard Limits

| Element                         | Limit            |
| ------------------------------- | ---------------- |
| Caption                         | 2,200 characters |
| Caption truncation ("More")     | 125 characters   |
| Hashtags (max allowed)          | 30               |
| Hashtags (recommended)          | 3–5              |
| Bio                             | 150 characters   |
| Stories per batch (recommended) | 5–7 at once      |

### Optimal lengths

- Feed caption: **125–180 characters** for max engagement
- First 125 characters must contain the hook/key message
- Reels caption: keep it short, hook in first line

### Algorithm signals (ranked by weight)

1. **Saves** — highest weight
2. **Shares** — high weight
3. **Comments** — medium-high weight
4. **Likes** — lower weight
5. **Watch time / completion** (Reels) — critical

### Content rules

- Carousels generate **35% more reach** than single images
- Reels get **22% more reach** than regular videos
- Instagram penalizes reposted/recycled content — originality rewarded
- First comment can hold hashtags (doesn't affect caption)
- Stories: 60% value-driven, 40% engagement-focused
- Use interactive stickers (polls, questions) in Stories strategically

### Post formats

- Feed post (single image, carousel, video)
- Reels (9:16, up to 90 seconds for feed; up to 15 min for longer)
- Stories (24h, 9:16, interactive features available)
- Live
- Broadcast Channel (one-to-many messaging)

### Audience & tone

- Visual-first platform
- Authentic over polished (2025 algorithm shift)
- Best for: lifestyle, brand aesthetics, product discovery, Reels virality

## LinkedIn

### Hard Limits

| Element                      | Limit              |
| ---------------------------- | ------------------ |
| Post                         | 3,000 characters   |
| Post truncation ("See More") | 140 characters     |
| Article                      | 125,000 characters |
| Article title                | 100 characters     |
| Comment                      | 1,250 characters   |
| Hashtags recommended         | 3–5                |

### Optimal lengths

- **1,000–1,500 characters** for highest engagement
- Hook must land in first **50 characters** (before truncation on mobile)
- Long-form articles: **1,900–2,000 words** perform best

### Algorithm signals

- **Early engagement** (first 60–90 min) critical for distribution
- **Comments > Likes** for reach amplification
- 2nd and 3rd degree network shares expand beyond followers
- Native content (no external links in post body) gets more reach
- Personal posts from individuals outperform brand page posts

### Content rules

- Links → put in **first comment**, not in post body
- Start with contrarian or unexpected hook
- End with a clear, easy-to-answer question
- Short paragraphs, one idea per paragraph
- Grade 5–7 reading level performs best
- Post 1–3 times per day maximum

### Post formats

- Text post
- Document / PDF carousel (high save rate)
- Image post (single or multi-image)
- Video (native performs better than YouTube links)
- Poll (up to 4 options, runs up to 2 weeks)
- Article (long-form, indexed by Google)
- Newsletter

### Audience & tone

- Professional, B2B, career-focused
- Thought leadership and industry insight content wins
- Personal stories with professional lesson → high engagement
- Best for: B2B lead gen, recruitment, professional authority

## Facebook

### Hard Limits

| Element              | Limit                        |
| -------------------- | ---------------------------- |
| Post (personal/page) | 63,206 characters            |
| Ad primary text      | 125 characters (recommended) |
| Ad headline          | 40 characters                |
| Ad description       | 30 characters                |
| Username             | 5–50 characters              |
| Page description     | 255 characters               |
| Hashtags recommended | 1–2                          |

### Optimal lengths

- **Under 80 characters** → 66% higher engagement
- Under 50 characters performs best for direct engagement
- For value-driven posts: 100–250 characters acceptable
- Video: under 3 minutes for highest completion; 15 sec for ads

### Algorithm signals

- **Meaningful interactions** (comments, shares) over passive likes
- Content that keeps users **on Facebook** is rewarded
- Friends/family content prioritized over pages
- Groups content gets significant organic boost
- Video watch time is a strong signal

### Content rules

- Avoid "engagement bait" (explicit "like if you agree" prompts — penalized)
- External links reduce reach — native content preferred
- Facebook Groups: best organic reach on the platform
- Conversational tone > promotional
- Posts under 80 characters + strong visual = best combo

### Post formats

- Text post
- Image post / album
- Video (native upload > YouTube link)
- Reels (9:16, up to 90 seconds)
- Stories (24h)
- Poll
- Event
- Group post

### Audience & tone

- Broad demographic, skews 35+
- Community and family-oriented
- Best for: local business, events, groups, broad awareness, retargeting via ads

## TikTok

### Hard Limits

| Element                   | Limit                               |
| ------------------------- | ----------------------------------- |
| Caption                   | 2,200 characters                    |
| Caption recommended       | 100–150 characters                  |
| Hashtags recommended      | 3–5                                 |
| Video length (standard)   | 15 sec – 10 min                     |
| Optimal video length      | 15–60 seconds (hook in first 3 sec) |
| Optimal for boosted posts | 9–15 seconds                        |

### Algorithm signals (ranked by weight)

1. **Watch time / completion rate** — most critical signal
2. **Rewatch rate** — very strong signal
3. **Shares** — high weight (new in 2025)
4. **Saves** — high weight (new in 2025)
5. **Comments** — medium weight
6. **Likes** — lowest weight (no longer dominant)

### Content rules

- Hook must land in **first 3 seconds** — no exceptions
- Aim for **70%+ watch time** on every video
- Niche-focused content outperforms generic viral attempts ("micro-virality")
- Follower count does NOT affect initial distribution — every video starts at zero
- Trending audio increases discoverability
- On-screen text improves both accessibility and algorithmic categorization (TikTok is becoming a visual search engine)
- Consistent niche posting helps algorithm identify and grow your community
- Quality over quantity: HD video preferred by algorithm
- Saves and shares now weighted above likes in 2025 update

### Post formats

- Short video (9:16, 1080×1920px)
- Photo carousel (disrupts feed scroll)
- TikTok Live
- Duet / Stitch (collaborative formats)
- TikTok LIVE

### Audience & tone

- Gen Z and Millennials primary
- Authentic, lo-fi content can outperform polished production
- Community and subculture driven (#BookTok, #FinanceTok, etc.)
- Best for: viral awareness, Gen Z reach, product discovery, community building

## YouTube

### Hard Limits

| Element           | Limit                               |
| ----------------- | ----------------------------------- |
| Title             | 100 characters                      |
| Title recommended | 60 characters (avoids truncation)   |
| Description       | 5,000 characters                    |
| Tags (combined)   | 500 characters                      |
| Shorts length     | Max 60 seconds (9:16, 1080×1920px)  |
| Standard video    | Up to 12 hours (verified accounts)  |
| Skippable ad      | Up to 6 min (skippable after 5 sec) |
| Non-skippable ad  | 15–20 seconds                       |
| Bumper ad         | 6 seconds                           |

### Algorithm signals

- **Click-through rate (CTR)** on thumbnail+title — critical
- **Watch time** and **average view duration** — most important
- **Shorts**: completion rate is the key metric
- Subscriber notifications and session starts matter
- Comments, likes, and shares are secondary

### Content rules

- Thumbnail + title must work together as a single unit
- First 30 seconds must earn continued watching
- Chapters (timestamps) improve retention and search
- Shorts see **25% higher completion rates** for ads
- End screens and cards extend watch session
- Consistent upload schedule builds algorithm trust
- Keywords in title, description, and tags — YouTube is a search engine

### Post formats

- Standard video (16:9)
- YouTube Shorts (9:16, max 60 sec)
- Live stream
- Premiere (scheduled video with live chat)
- Community post (text, poll, image)

### Audience & tone

- Broadest age range of all platforms
- Educational, entertainment, and how-to content dominates
- Best for: SEO, long-form authority, tutorial content, product reviews

## Reddit

### Hard Limits

| Element    | Limit                                        |
| ---------- | -------------------------------------------- |
| Post title | 300 characters                               |
| Post body  | 40,000 characters                            |
| Comment    | 10,000 characters                            |
| Username   | 3–20 characters                              |
| Hashtags   | Not used (Reddit uses subreddit communities) |

### Optimal lengths

- Title: **60–120 characters** — specific, intriguing, invites response
- Body: as long as needed, but front-loaded with value
- Comment: concise and direct — Redditors downvote fluff

### Algorithm signals

- **Upvotes in first 30–90 minutes** — critical for subreddit ranking
- **Comment velocity** early after posting
- Subreddit-specific rules and culture compliance
- Karma history affects credibility

### Content rules

- **Lurk before posting** — understand each subreddit's culture
- Transparency is mandatory: declare brand affiliation clearly
- Never use corporate language ("industry-leading", "cutting-edge", "unlock your potential")
- Write in **second person** ("you") not third person ("users")
- End post with a **strong, specific question** to drive comments
- Links to gated content → instant negative reaction
- Self-promotion without community value → downvotes and bans
- Respond to comments within 48–72 hours of posting

### Post formats

- Text post (most common for discussion)
- Link post
- Image / Gallery post
- Poll
- AMA (Ask Me Anything)
- Crosspost (share to multiple subreddits — use sparingly)

### Audience & tone

- Highly skeptical of marketing
- Rewards authenticity, expertise, vulnerability
- Community-first: add value before promoting
- Best for: niche communities, research/feedback, organic credibility, thought leadership

## Cross-Platform Rules

### Universal best practices

**Links**

- X/Twitter → in first reply
- LinkedIn → in first comment
- Facebook → reduces reach, use sparingly
- Instagram → bio only (or Stories link sticker)
- Threads → in reply when possible
- TikTok → in bio
- Reddit → only if genuinely useful and ungated
- YouTube → in description and pinned comment

**Hashtags by platform**
| Platform | Recommended # |
|---|---|
| Instagram | 3–5 |
| TikTok | 3–5 |
| LinkedIn | 3–5 |
| X/Twitter | 1–2 (last tweet in thread) |
| Facebook | 1–2 |
| Threads | 1–3 |
| YouTube | In description, keyword-focused |
| Reddit | Not applicable |

**Emojis**

- Work well: TikTok, Threads, Instagram, Facebook
- Use sparingly: LinkedIn (guide structure, not decorate)
- Avoid: Reddit (often seen as performative)

**Cross-posting**

- Never copy-paste the same text across platforms
- Always rewrite natively for each platform's tone and format
- Repurpose the _idea_, not the _copy_

### Platform selection guide

| Goal                       | Best platform(s)                 |
| -------------------------- | -------------------------------- |
| Viral awareness / Gen Z    | TikTok, Instagram Reels          |
| Thought leadership         | LinkedIn, X/Twitter              |
| Community building         | Reddit, Facebook Groups, Threads |
| SEO / long-form authority  | YouTube, LinkedIn Articles       |
| Real-time conversation     | X/Twitter, Threads               |
| Product discovery / visual | Instagram, TikTok, Pinterest     |
| B2B lead generation        | LinkedIn                         |
| Broad reach / all ages     | Facebook, YouTube                |
