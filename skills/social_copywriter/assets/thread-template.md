## Technical checklist before publishing

- [ ] Each post within character limit (500 Threads / 280 X)
- [ ] Every post works as a standalone unit
- [ ] At least 1-2 cliffhangers in the body posts
- [ ] No links inside the thread body
- [ ] Hashtags only in last post
- [ ] Hook does not start with a generic opener
- [ ] Closing post has a clear question or CTA
