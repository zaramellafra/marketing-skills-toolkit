# Format Writing Instructions

Detailed copywriting rules for each social media text format.
Read the relevant section before writing any output.

---

## 1. Single Post

A standalone post. The most common format across all platforms.

### Structure

```
[Hook — first line, stops the scroll]
[Body — develops the idea, delivers value]
[Closing / CTA — what the reader should do or think next]
[Hashtags — per platform rules from skill-platform-knowledge]
```

### Hook types (choose based on objective)

- **Bold statement** — Strong or counter-intuitive claim
- **Surprising statistic** — Data that creates cognitive tension
- **Direct question** — Opens an information gap
- **Value promise** — States exactly what the reader gets
- **Common mistake** — Identifies a widespread problem
- **Personal story** — Starts in medias res

### Rules

- First line must work on its own — it's what appears before "read more"
- One idea per post — never two different messages in the same post
- Clear but non-aggressive CTA — invite, don't push
- Vary sentence rhythm: short sentences mixed with slightly longer ones

## 2. Thread

A series of connected posts that develop an idea progressively.
Used on: X/Twitter, Threads, LinkedIn (as a post series).

### Structure

```
Tweet 1 — Hook (most important — determines if the thread gets read)
Tweet 2 — Setup / context (why this matters)
Tweet 3-N — Progressive development (each tweet = one point, one insight, one step)
Second-to-last tweet — Main lesson / takeaway
Last tweet — CTA + hashtags
```

### Hook tweet (Tweet 1) — special rules

- Must work as a standalone post
- Use one of the hook types from Single Post
- Explicitly promise what the reader will find in the thread

### Body tweets (Tweet 2-N)

- One idea per tweet — never two concepts in the same tweet
- Each tweet must stand alone (the algorithm may surface individual tweets)
- Cliffhanger every 1-2 tweets to maintain reading momentum
- Use numbering for listicles: "1/ ...", "2/ ..."
- Short sentences. White space. Easy to scan.

### Closing tweet

- Summarize the main takeaway in 1-2 sentences
- Soft CTA: follow, save, share, comment
- Hashtags (1-2, last tweet only)
- Links (in a reply to the thread, never inside the thread itself)

### Recommended length

- Minimum: 5 tweets / Optimal: 7 tweets / Maximum recommended: 10 tweets

## 3. Carousel

Slide-by-slide text script. The designer applies visuals on top.
Used on: LinkedIn, Instagram, Facebook, TikTok (photo carousel mode).

### Structure

```
Slide 1 — Cover: strong hook, clear promise
Slide 2 — Problem or context (why this is relevant)
Slide 3-N — Development (each slide = one point, one step, one insight)
Second-to-last slide — Takeaway or summary
Last slide — Explicit CTA
```

### Rules per slide

- One idea per slide — never overload
- Headline required — short, readable even without the body
- Body optional — max 1-3 lines, expands the headline
- Continuity — end of each slide should invite the reader to swipe
- Use slide numbers for listicles: "3 of 7"

### Cover slide (Slide 1) — special rules

- Headline: max 8-10 words, immediate impact
- Optional subtitle: specifies the promise
- Avoid vague: "Tips to grow" ❌ / Prefer specific: "5 mistakes blocking your LinkedIn growth" ✅

### CTA slide (Last slide)

- One action only
- Examples: "Save this post" / "Follow for more on [X]" / "Comment with [Y]"

### Recommended length

- Minimum: 4 slides / Optimal: 6-8 slides / Maximum: 10-12 slides

### Output format for carousels

```
SLIDE 1 — COVER
Headline: [text]
Subtitle: [optional]

SLIDE 2
Headline: [text]
Body: [text]

[...continue for each slide...]

SLIDE N — CTA
Headline: [text]
CTA: [action]
Designer note: [optional]
```

## 4. Video Caption (Reels / TikTok / YouTube Shorts)

The text accompanying a video. It completes the video — it does not describe it.

### Structure

```
[Hook — first line, must push the viewer to watch]
[1-2 lines of context or teaser — optional]
[CTA — what to do after watching]
[Hashtags — per platform rules]
```

### Platform-specific rules

- **Instagram Reels**: first line critical (visible in feed), max 2,200 chars, 3-5 hashtags
- **TikTok**: 100-150 chars optimal, 3-5 hashtags, include keywords for search
- **YouTube Shorts**: brief, supports the video title

### Hook types for video captions

- Content teaser: "The trick no course ever teaches you 👇"
- Question the video answers: "Why does your engagement drop on weekends?"
- Bold claim: "I tried this strategy for 30 days. The result:"
- Behind the scenes: "What happens before I hit publish"

## 5. Stories Copy

Text for Instagram/Facebook Stories. Short, direct, mobile-first.

### Rules

- Max 1-2 lines of visible text
- Use questions for interactive Stories (poll, question sticker)
- Most informal tone of all formats
- One CTA only, clear and direct
- For Story series (5-7 cards): each card works alone but invites to the next

### Story types

- **Teaser** / **Behind the scenes** / **Poll-Question** / **Announcement** / **Reminder**

## Universal Narrative Frameworks

Choose the framework that best fits the content objective.

| Framework                                             | Best for                                        |
| ----------------------------------------------------- | ----------------------------------------------- |
| **PAS** (Problem → Agitate → Solution)                | Conversion, products, services                  |
| **AIDA** (Attention → Interest → Desire → Action)     | Launches, announcements, ads                    |
| **Storytelling** (Situation → Conflict → Resolution)  | Case studies, personal stories, brand narrative |
| **Listicle** (numbered list of insights/tips)         | Educational content, saves, threads             |
| **Contrarian** (challenge a common assumption)        | Thought leadership, engagement, discussion      |
| **How-to / Tutorial** (step-by-step)                  | Carousels, threads, educational video captions  |
| **Data + Insight** (surprising stat → interpretation) | Authority building, LinkedIn, X/Twitter         |

### PAS example

```
P: "You post every day but you're not growing."
A: "Every hour spent writing is a wasted hour if no one reads it."
S: "The problem isn't frequency. It's structure. Here's how to fix it."
```

### Contrarian example

```
"Everyone tells you to post every day. I'm telling you to stop."
```

### Data + Insight example

```
"94% of threads get fewer than 10 retweets. Here's why yours won't."
```
