# Marketing Skills for AI Agents

> **Status: 🚧 Work in Progress**  
> This repository is actively being built. Skills are being added and refined. Contributions and feedback are welcome.

---

A public collection of marketing skills for AI agents — compatible with **Claude**, **Gemini CLI**, and **ChatGPT**.

Each skill is a structured framework that guides an AI agent through a specific marketing process, producing consistent, actionable outputs.

---

```
├── skills/
│   ├── positioning/

└── chatgpt/
    └── growth-marketing-system-prompt.md
```

---

## Compatibility

| Platform       | How to use                                                |
| -------------- | --------------------------------------------------------- |
| **Claude**     | Place skill folders in your project's `skills/` directory |
| **Gemini CLI** | Copy skill folders into `~/.gemini/skills/`               |
| **ChatGPT**    | Use the system prompt in the `chatgpt/` folder            |
